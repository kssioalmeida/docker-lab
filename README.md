# Docker-composer lab

Este é um repositórios de estudos utilizando docker.

##### Requesitos:

* [Docker] 
* [Docker-compose] 


```bash
git clone https://github.com/kssioalmeida/docker-lab.git
cd docker-lab
docker-compose up -d
```

Acesse a aplicação

```bash
localhost:5000
```

[docker]: <https://www.docker.com/>
[docker-compose]: <https://docs.docker.com/compose/>
